# Shoe Website Test

React + Vite product landing page with:

- square featured and sport product cards
- hover zoom/rotation product interactions
- Training Studio banner hover video

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Shareable site

This repo is configured for GitHub Pages deployment from the `main` branch via GitHub Actions.

Expected Pages URL:

`https://pragyan-divami.github.io/Shoe_website_Test/`
